The BlackArch Linux community appears on several places in the Internet. The following listing shows you where we are active and gives you detailed information about the activity in each place. In case of a direct query you can use our e-mail address: blackarchlinux[at]gmail.com.

BlackArch Linux Github Repository

Every work (literally every bit) of BlackArch Linux is on our github. You can basically watch and track each of our work for BlackArch. We don't hide anything and everything is transparent for you.

It is the recommended and official way to contribute to BlackArch in any kind of forms. So, feel free to post questions and suggestions on our github issue tracker or open pull requests for version upgrades or new tools. We also accept pull requests of all sizes.


BlackArch Linux Twitter Feed

We use mainly twitter to tweet about new tools we added, new mirrors we have got and few other things related to BlackArch Linux. If you don't like browsing and staying on websites you can instead use your twitter client and follow us to get BlackArch news.



BlackArch Linux IRC Channel

We use IRC to discuss about BlackArch but also about social things. Also, you would see any changes made on our github repositories through notification-bot. That way, you would immediately see if a new tool has been released or any upgrades has been performed, etc.

You can find the official BlackArch Internet Relay Chat channel here:
#blackarch on freenode IRC servers.


BlackArch Linux Blog & News

Blog entries and news about BlackArch Linux can be found on our blog site. The content is mostly a more detailed version of our twitter feeds, but you will also find news about new ISO releases and other news.

We also release interesting blog entries about BlackArch Linux, hacking and security written not only by our developers. If you want to add a blog entry feel free to open a pull request on our blackarch-site repository.
