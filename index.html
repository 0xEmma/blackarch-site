About BlackArch Linux

BlackArch Linux is an Arch Linux-based distribution for penetration testers and security researchers. The repository contains 1191 tools. You can install tools individually or in groups. BlackArch Linux is compatible with existing Arch installs. For more information, see the installation instructions. Also, news are published on our blog site.

Please note that BlackArch is a relatively new project. To report bugs and request new tools, please visit the issue tracker on Github, stop by IRC, or email us.
